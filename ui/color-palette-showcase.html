Design Guidelines

Red (#E31837): Use for primary branding, headers, important CTAs, and key UI elements that need to stand out.

Orange (#FF6F40): Use for warnings, medium-priority alerts, and informational elements that require attention but aren't critical.

Green (#7FCC72): Use for success states, add buttons, positive feedback, and completed actions.

Accessibility: All color combinations maintain WCAG AA standards for contrast ratios, ensuring readability for all users.
